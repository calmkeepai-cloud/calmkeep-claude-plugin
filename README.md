# Calmkeep Claude Plugin

Calmkeep is a continuity layer that preserves architectural and reasoning stability across long multi-turn workflows.

This plugin allows Claude Code to invoke Calmkeep automatically during complex development tasks.

## Install

Add the Calmkeep marketplace:

/plugin marketplace add calmkeepai-cloud/calmkeep-claude-plugin

Install the plugin:

/plugin install calmkeep-continuity

## Requirements

Install the Calmkeep SDK:

pip install calmkeep

Set your Calmkeep API key:

export CALMKEEP_API_KEY=your_key_here

You can obtain an API key at:

https://calmkeep.ai
